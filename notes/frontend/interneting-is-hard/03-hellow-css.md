# Units of Measurement

## Absolute - px

## Relative

em relative to the element's font size. if parent = 20px, then 1em = 20px
rem relative to the root element's font size (usually the <html> tag). if root = 16px, then 1rem = 16px

vh / vw are relative to the viewport's height / width, respectively
The visible area of the screen, 1vw = 1% of the viewport width

% relative to parent element's font size
ch relative to the width of the "0" (zero)
