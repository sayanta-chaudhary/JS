# Optimize for legibility

Fonts for headlines usually have tigher `litter-spacing` and shorter lowercase letters
While fonts for smaller sized have wider `litter-spacing` and taller lowercase letters

Baseline, not center: When you use multiple font sizes `align-items: center`

Don't center long form text
Center-alignment can look great for headlines or short, independent blocks of text
But if something is longer thatn two or three lines, it will almost always look better left-aligned
Right-align numbers - if you're designing a table that includes numbers, right-align them
