What sets a good design apart from a bad one, is the use of space.
Any size or space you use must be divisible by 4 (Consistency = Good UX)

# Margin, Padding, and Gap

- Always apply the margin to the bigger text elements
- Never use margins on reusable components
- Single element — Use `margin`
- Group of elements — Use `gap`

Start with too much white space, then remove it until you're happy with the result

# Laws of spacing

- Law of Proximity
- Law of Common Region
- The 8pt Grid System
- Use consistency spacing scale, Never pick arbitrary values

# Establish spacing & sizing system

```plaintext

## Group elements
1 = 0.25rem = 4px
2 = 0.5rem = 8px - 💡
3 = 0.75rem = 12px

# Used for padding & grouping buttons
4 = 1rem = 16px - 💡

## Seperate elements
You don't really have to skip levels once you get pass 1rem
6 = 1.5rem = 24px - 💡
8 = 2rem = 32px
16 = 4rem = 64px
```

# Layout

## You don't have to fill the whole screen 💡

- Just because you have the space, doesn't mean you need to use it
- Give each element just the space it needs, don't make something worse
  just to make it match something else
- You shouldn't try to cram everything into a small area unnecessarily either
